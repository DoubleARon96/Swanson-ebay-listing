Full Description

The original Swanson® Speed® Square has been an indispensable tool for tradesmen for nearly a century. Swanson’s Speed® Trim Square has all the great features you’ve always counted on, like the patented 3½ in. Diamond™ mark and fade-resistant easy-read markings but we’ve taken it up a notch. To make accurate layouts even easier, there’s a scribe bar with guides milled every 1/8 in. Also, drilling holes centered on 3/4 in. stock couldn’t be simpler thanks to a hole milled 3/8 in. from the inside edge, right-sized for self-centering drill bits. It's built to last a lifetime, prou`dly made in the USA with high-grade aluminum alloy construction. The new Swanson Speed Trim Square’s convenient fit, features, and patent pending design make it a truly must-have tool for cabinetry, moulding, trim and so much more.
